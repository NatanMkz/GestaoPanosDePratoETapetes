Necessário para poder executar o projeto

1 - Necessário windows (recomendamos o windows 10) pois é um projeto Wpf, que é exclusivamente para as aplicações windows.
2 - Necessário Visual Studio instalado, e também é necessário incluir na instalação o item "Desenvolvimento para Desktop com o .NET"
3 - instalar nos pacotes NuGet o pacote "Microsoft.EntityFrameworkCore.Sqlite.NetTopologySuite"
4 - Se divertir! XD
